# Safety

DupClean is designed with safety as a priority to prevent accidental data loss.

- **Trash/Recycle Bin:** All removed files are moved to the system Trash/Recycle
  Bin, not permanently deleted. You can restore them easily.
- **Ignore Hidden Files:** Hidden files and `.DS_Store` files are automatically
  ignored to prevent system damage.
- **Preview:** You can preview audio files before choosing to remove them.
- **Safe Targets:** Cache cleaning targets that could risk system stability are
  not selected by default.
